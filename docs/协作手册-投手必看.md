# 投手协作手册（v2 · 2026-05-14 改）

**给：** HZM / CHJ / HNN / ZXR / LZL / PLZ

**不需要 GitHub 账号。不需要装任何软件。浏览器打开就用。**

---

## 一句话流程

```
打开 URL → 选我是谁 → 拖 xlsx 上传 / 点 agent 按钮 → 5-10 分钟看自己看板
```

## 1. 入口

```
https://multisite-platform.pages.dev/
```

（实际 URL 以 ZJB 发的为准）

首次打开：
- 选 owner（你是谁，例如 HZM）
- 填 ZJB 发的访问口令
- 浏览器记住，之后不用再填

## 2. 三件事

### A. 看自己组数据
打开 URL → 点上方导航的自己（HZM/CHJ/...）→ 看板自动加载

### B. 上传 FB xlsx
打开 URL → 点 **→ 投手工作台** → 把 xlsx 拖到虚线框，或点击选择
- 系统自动按 `{你}-{日期}.xlsx` 重命名
- 看到 "✓ 已上传 · 处理中" 就好了
- 5-10 分钟后回看板看新数据

### C. 触发 agent 跑（不上传文件，按需要点）
工作台下方 "② 触发 agent" 区，11 个按钮：

| 常用 4 个 | 干什么 |
|---|---|
| 数据分析 | 看灯色 / CPHQ / HVU |
| 归因诊断 | ROI 异动排外因 |
| 知识沉淀 | 提炼命题 |
| 策略简报 | 命题→投放方向 |

点一下 → 看到 "✓ 已提交" → 5-10 分钟刷新看板

## 3. 处理时间

- 上传 → ZJB 后台 GitHub Actions 自动跑（不需要 ZJB 在）
- 平均 5-7 分钟
- 看板 URL 刷新即可看新结果

## 4. 文件名怎么命名

不用管。系统自动改成 `{你}-{日期}.xlsx`。
你拖的文件叫 `广告组数据.xlsx`，系统会保存成 `HZM-2026-05-14-广告组数据.xlsx`。

## 5. 出错了怎么办

| 看到 | 原因 | 怎么办 |
|---|---|---|
| 口令错误 | 访问口令填错 / ZJB 改了 | 找 ZJB 要新口令 |
| 不是 xlsx | 你拖的是 csv / pdf | 在 FB 后台导出 xlsx 格式 |
| owner 不在白名单 | 选的 owner 拼错 | 重选下拉框 |
| GitHub 502 | 后端临时抖动 | 等 1 分钟重试 |

## 6. 隐私

xlsx 数据进 ZJB 的后台 repo（私有/公开看 ZJB 配置）。
**你能看到自己的，看不到别人的。**
